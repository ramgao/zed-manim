# YouTube 4K Template

A Manim starter project for 3840x2160, 60 FPS intro animations.

Use the low-quality task while iterating, then render the 4K task for final output.

## Set Up

```sh
python -m venv .venv
.venv/bin/python -m pip install manim
```

On Windows PowerShell:

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install manim
```
