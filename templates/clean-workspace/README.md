# Clean Workspace Template

A beginner-friendly Manim project for Zed.

It includes:

- `main.py` with a simple renderable scene
- `pyrightconfig.json` for basedpyright
- `pyproject.toml` for Ruff
- `.zed/tasks.json` for render tasks

## Set Up

```sh
python -m venv .venv
.venv/bin/python -m pip install manim
```

On Windows PowerShell:

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install manim
```

Open this folder in Zed and select the `.venv` Python toolchain if needed.
