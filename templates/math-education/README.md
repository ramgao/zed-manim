# Math Education Template

A Manim starter project for lessons with equations, axes, graphs, and highlighted results.

This template uses `MathTex` when LaTeX is available and falls back to plain `Text` otherwise, so the starter scene can still render in a fresh environment.

## Set Up

```sh
python -m venv .venv
.venv/bin/python -m pip install manim
```

On Windows PowerShell:

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install manim
```
