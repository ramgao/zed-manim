# Example Scenes

Additional Manim learning examples live here.

Render them from this directory:

```sh
manim -pql parabola_3d.py Parabola3D
manim -pql quadratic_solution.py QuadraticFormulaDerivation
```

`quadratic_solution.py` uses `MathTex`, so it requires a working LaTeX installation such as MiKTeX.

Generated `media/` files should not be committed.
